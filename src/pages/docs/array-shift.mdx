## Basic usage

Given a list of items, return an array that shift right n positions.

```ts
import { shift } from 'radash'

const arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]

shift(arr, 3) // => [7, 8, 9, 1, 2, 3, 4, 5, 6]
```
